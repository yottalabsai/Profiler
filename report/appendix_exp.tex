\subsection{Workload Reproduction}
\label{sec:appendix:repro}

All six workloads are reproduced using the commands below.
Each workload requires an NVIDIA GPU (Ampere, Hopper, or Blackwell),
\nsys $\ge$ 2025.1, \ncu $\ge$ 2025.4.1, PyTorch 2.11, and CUDA 12.x.

\begin{lstlisting}[style=pystyle]
# Install
pip install -e .

# Capture baseline profile for one workload
operator-profiler profile scripts/workloads/transformer_block.py \
    --output runs/transformer_block/baseline \
    --warmup-iters 2

# Collect hardware counters
operator-profiler map runs/transformer_block/baseline.manifest.json \
    --script scripts/workloads/transformer_block.py \
    --ncu-sudo --output runs/transformer_block/profile.json

# Run all six workloads in batch
python scripts/run_all_profiles.py
\end{lstlisting}

Profiling results, optimized workload scripts, and \texttt{profile.json} /
\texttt{profile\_optimized.json} pairs for all six examples are in
\texttt{examples/} in the repository.  Each subdirectory contains
\texttt{report.md} with the measured before-and-after hardware counter values.

\subsection{Additional Limitations}
\label{sec:appendix:limitations}

Beyond the limitations noted in \secref{sec:conclusion}:
\begin{itemize}[leftmargin=1.5em]
  \item \textbf{\ncu replay timing inflation.}  All \texttt{duration\_ns} values in
    \texttt{profile.json} reflect \ncu's serialized replay execution, which runs
    2--5$\times$ slower than real wall-clock execution.  Values should be used for
    relative comparison only; \texttt{torch.cuda.Event} timing is authoritative for
    absolute latency.
  \item \textbf{Multi-GPU attribution.}  The current pipeline profiles a single GPU.
    Distributed workloads with NCCL communication require per-stream, per-device
    profiling runs; cross-device attribution is not yet supported.
  \item \textbf{Dynamic shapes.}  Shape validation enforces that \ncu replay inputs
    match the \nsys capture.  Models with truly dynamic shapes (varying sequence
    length per call) require separate captures per shape bucket.
\end{itemize}
